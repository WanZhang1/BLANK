# BLANK


人之初，一如水之清澈透明。所谓BLANK，正是此意。借由书籍，净化心灵，提升修养。


## Intro

个人所读书籍没有什么种类限制，是好书都会去读，偏爱心理学、经济学、哲学，文学类也会涉猎，这些部分的读书笔记你可以在年份文件夹下找到相关描述，而技术书籍会统一放在tech-books文件夹下面，可能并不会有相关笔记描述，这里只提供一些pdf版本电子书籍。另外，个人读书笔记较为浅显，仅供参考！
